## Replacing Images

If you want to use your own images:

1. **Add images to `frontend/public/images/` folder**
2. **Update the image paths in the components:**

```tsx
// Instead of:
src="https://images.unsplash.com/photo-..."

// Use:
src="/images/your-image.jpg"
```

## Recommended Image Sizes

- Hero images: 1200x600px
- Feature cards: 400x300px
- Mission cards: 400x250px
- Header banners: 800x400px
