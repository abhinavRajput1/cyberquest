# Images Directory

Place your custom images here. The application will automatically use these images instead of the Unsplash placeholders.

## Required Images

### Landing Page
- `hero-cybersecurity.jpg` - Hero section image (1200x600px recommended)
- `phishing-feature.jpg` - Phishing missions feature card (400x300px)
- `network-feature.jpg` - Network puzzles feature card (400x300px)
- `osint-feature.jpg` - OSINT challenges feature card (400x300px)
- `cta-cybersecurity.jpg` - Call-to-action section image (600x400px)

### Authentication Pages
- `login-cybersecurity.jpg` - Login page side image (600x800px)
- `signup-network.jpg` - Signup page side image (600x800px)

### Dashboard
- `phishing-card.jpg` - Phishing missions card (400x250px)
- `network-card.jpg` - Network puzzles card (400x250px)
- `osint-card.jpg` - OSINT challenges card (400x250px)

### Missions Page
- `missions-header.jpg` - Missions page header banner (800x400px)

### Mission Detail Pages
- `mission-phishing.jpg` - Phishing mission detail header (800x400px)
- `mission-network.jpg` - Network mission detail header (800x400px)
- `mission-osint.jpg` - OSINT mission detail header (800x400px)

## Image Formats
- Supported: JPG, JPEG, PNG, WebP
- Recommended: JPG or WebP for smaller file sizes

## Naming Convention
- Use lowercase letters
- Separate words with hyphens (-)
- Keep file names descriptive

## Example
```
public/images/
├── hero-cybersecurity.jpg
├── phishing-feature.jpg
├── network-feature.jpg
├── osint-feature.jpg
├── cta-cybersecurity.jpg
├── login-cybersecurity.jpg
├── signup-network.jpg
├── phishing-card.jpg
├── network-card.jpg
├── osint-card.jpg
├── missions-header.jpg
├── mission-phishing.jpg
├── mission-network.jpg
└── mission-osint.jpg
```

## After Adding Images
Once you've added your images, the application will automatically use them instead of the Unsplash URLs. No code changes needed if you use the exact file names listed above.
